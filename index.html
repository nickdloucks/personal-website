Roman Numeral Conversion Tool

This is a reference tool that converts a base-10 number to a Roman Numeral string.

It truncates (rounds down) any floating-point numbers to an integer before conversion. The only parameter is an integer in base-10 format. The return value is either a Roman Numeral or error message for invalid inputs. Process: 1. convert decimal to string, split into an array of characters, then reverse the array. 2. loop thru the reversed array of chars; inside the loop, store the digit as a number for comparison. 3. assign the correct sequence of small, medium, and big letters based on the digit's place in the base-10 input. 4. push the letter sequence onto the final roman numeral string and return the finished string.
